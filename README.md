# React Movie App

## Description
Movie browsing application built with React, featuring routing, filtering and dynamic UI components.

## Features
- Movie listing
- Filtering movies
- Rating system
- Navigation with React Router
- Component-based UI

## Tech Stack
- React
- JavaScript
- React Router
- Bootstrap

## Purpose
This project demonstrates my ability to build frontend applications using React with routing and component-based architecture.
